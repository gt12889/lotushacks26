# Frontend (Next.js)

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

Megladon MD uses **`npm run dev`** on port **3005** (see root `frontend/package.json`).

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3005](http://localhost:3005).

You can start editing the app under `src/app`. This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) for Geist where configured.
